# :iphone: 모두의 조경

조경학과 학도들을 위한 교내 시설물과 수목들에 대한 관심과 지식을 함양시키는 ‘캠퍼스 조경관리 어플리케이션’입니다. <br>
학과 학생들이 교내에 어떤  수목이나 시설물이 있는지 알 수 있도록 도와주고, 관리가 잘 안 되어있을 시 메모를 남김으로써 빠른 조치를 할 수 있도록 합니다.
